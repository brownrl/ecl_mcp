ECL Template Main Content

This template includes all major ECL components: site header with logo, menu, language selector, and search; page header with breadcrumbs and description; main content area; and footer with logo, social media, and columnar links.

Getting Started

To use this template:

  1. Replace placeholder content with your actual content
  2. Update the site name, logo, and navigation links
  3. Modify breadcrumbs and page titles for each page
  4. Ensure all asset paths are correct for your setup
  5. Test responsiveness on different screen sizes

ECL Components Used

  • Site Header: Logo, language selector, search form, banner, menu
  • Page Header: Breadcrumb navigation, page description
  • Site Footer: Logo, description, social media links, columnar link sections
  • Icons: SVG sprites with xlink:href references
  • Auto-initialization: ECL components activate automatically via ECL.autoInit()
